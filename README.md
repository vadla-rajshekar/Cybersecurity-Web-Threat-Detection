# Cybersecurity: Suspicious Web Threat Interactions

## Project Overview
This project focuses on detecting suspicious web traffic using machine learning techniques.
The dataset is collected from AWS CloudWatch logs.

## Objectives
- Analyze web traffic patterns
- Detect suspicious interactions
- Apply machine learning for threat detection
- Visualize attack behavior

## Tools & Technologies
- Python
- Google Colab
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Machine Learning Model
- Random Forest Classifier

## Results
- Achieved high accuracy in detecting suspicious web traffic
- Identified abnormal traffic patterns and attack sources

## Future Scope
- Real-time deployment
- Deep learning models
- Integration with SIEM tools
